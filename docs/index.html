Інпути та форми


Якщо до інпута додати data-required атрибут то це поле становитсья обов'язковим для заповнення. При пустому полі і відправці форми буде додаватись помилка


data-required також робить валідацію за значеннями:
1. data-required="email" - для інпутів вводу email
2. data-required="sms" - для інпуту вводу коду з SMS.



Додавання до форми атрибута data-goto-error надасть можливість прокручувати до першого поля з помилкою

Поля з атрибутом readonly: батьківським елементам і самому інпуту додаються класи _readonly, через що змінюються стилі полей. Якщо є кнопки очищення поля їм додається атрибут disabled="true". Якщо label має "підказки" то до них застосовується display: none

В проєкті налаштовано 2 мови (ru, uk) для перекладу тексту дропдауна з англійської. Переклад працює на основі вибраної мови сайту; мова за замовчуванням - англійська. Щоб додати необхідну мову перекладу, необхідно виконати налаштування у app.js (intlTelInput()). Доступно 37 мов. На сторінках, які використовують інтуп міжнародних телефонів файли плагіну підключені через тег script внизу сторінки
